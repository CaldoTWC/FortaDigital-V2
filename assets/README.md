# Assets Directory

Este directorio está preparado para almacenar los recursos visuales del sitio web FortaGuard.

## Estructura Recomendada

```
assets/
├── images/
│   ├── logo/          # Logo de FortaGuard
│   ├── team/          # Fotos del equipo
│   ├── icons/         # Iconos personalizados
│   └── backgrounds/   # Imágenes de fondo
├── documents/         # PDFs y documentos descargables
│   ├── BalancedScorecard_FortaDigital.pdf
│   ├── CasoNegocio_FortaDigital.pdf
│   ├── CedulaDeServicio_FortaDigital.pdf
│   ├── EstudioFactibilidad_FortaDigital.pdf
│   └── PracticaDisponibilidad_FortaDigital.pdf
└── README.md
```

## Notas

- Las imágenes deben estar optimizadas para web (formato WebP recomendado)
- Tamaño máximo recomendado: 500KB por imagen
- Los documentos PDF deben ser accesibles y con tamaño optimizado
- Se utiliza Font Awesome para iconografía general
